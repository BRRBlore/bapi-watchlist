# Bapi's Personal Watchlist Tracker

Automated watchlist for growth stocks at value prices.

**Thesis:** Buy quality growth companies (mid/small cap) during price corrections. Sell when price recovers to fair value.

**Criteria:**
- Quality Score ≥ 60 (confirmed growth stock)
- Discount to Fair Value ≥ 20% (value zone entry)
- Market Cap ₹500–50,000 Cr

**Data:** Reads from V2 Smart Stock Ranker daily — no separate scraping needed.
